Tout savoir sur les boucles

Qu'est-ce qu'une boucle, pourquoi a-t-elle été créée et comment l'utiliser ? Ce sont des questions que de nombreux débutants en programmation se posent.
Dans cette article, je vais essayer de détailler certains points.

Qu'est-ce qu'une boucle

Une boucle est une structure de contrôle en informatique qui permet de répéter un ensemble d'instructions plusieurs fois, généralement tant qu'une condition spécifiée est vraie, en informatique, il s'agit d'une valeur booléenne true. Cela permet d'automatiser l'exécution d'instructions sans avoir à les écrire plusieurs fois.

Pourquoi l'existence des boucles

On s'est rendu compte qu'écrire des instructions une à une pouvait être rébarbatif lorsque les valeurs de ces instructions étaient différentes, mais que le code restait le même. Les conséquences étaient que nous étions plus lents à coder nos applications et que nous étions gênés par la quantité de code qu'il y aurait pu y avoir si nous devions gérer des dizaines ou des centaines d'instructions pour chaque valeur variée.

Il est donc parfois utile d'exécuter un bloc de code plus d'une seule fois. Dans ce but, les langages de programmations proposent plusieurs types de boucles, qui parcourt le code à l'intérieur la boucle jusqu'à la fin et recommence immédiatement du début de la boucle. Dans le jargon informatique on va souvent appeler ça une itération, itérer à chaque tour de la boucle.
